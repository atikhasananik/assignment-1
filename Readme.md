# DevConf 2026 Landing Page

A modern conference landing page built with **HTML5** and **CSS3** for a professional developer event. This project showcases a responsive event website design featuring a hero section, speaker profiles, pricing plans, event schedule, and footer.

---

## 🚀 Project Summary

This assignment is a **static web page** for a tech conference called **DevConf 2026**. The design focuses on:

- Clean layout and strong visual hierarchy
- Clear call-to-action buttons
- Speaker showcase and ticket pricing
- A polished schedule section
- Simple and professional modern styling

---

## 🧩 What I made

- **Landing page hero section** with a bold headline and register button
- **Navigation bar** with links to speakers, schedule, venue, and blog
- **Speakers section** with profile cards for conference presenters
- **Pricing section** with three distinct ticket options
- **Schedule section** showing key sessions and a break event
- **Footer section** with logo, social icons, and copyright

---

## 🛠️ Technologies used

- **HTML5** for the page structure
- **CSS3** for styling, layout, and responsive design
- **Google Fonts** for custom typography
- **SVG icons** for social links in the footer

---

## 📁 Project files

- `index.html` — main page structure
- `style.css` — styling for layout, cards, buttons, and responsive behavior
- `assets/` — image and icon assets used throughout the page
- `DevConf2026.fig` — design/source file (optional)

---

## 💡 Notes

- The page is designed to look polished and modern with strong blue accent styling.
- The CSS uses reusable button styles and simple grid layouts for speaker/pricing cards.
- Subtle animations are included for hover states, card movement, and visual feedback.
- The page structure is easy to update for future sections like venue details or workshops.

---

## 📌 How to use

1. Open `index.html` in your browser.
2. Check the layout on desktop and mobile screen sizes.
3. Update text, images, or links inside `index.html` as needed.
4. Modify styles in `style.css` to adjust colors, spacing, or typography.
